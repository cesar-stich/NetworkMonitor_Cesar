# 📡 Network Health Monitor - UNI

Este es un proyecto modular en Python diseñado para monitorear la conectividad de red de servidores críticos. Es una herramienta útil para diagnósticos rápidos de infraestructura de telecomunicaciones.

## 🚀 Características
- **Arquitectura Modular**: Separación de lógica de red, utilidades y ejecución principal.
- **Monitoreo Silencioso**: Utiliza comandos del sistema para verificar estados sin saturar la terminal.
- **Formateo Visual**: Salida limpia con indicadores de estado (ONLINE/OFFLINE).

## 📂 Estructura del Proyecto
- `main.py`: Punto de entrada del programa.
- `scanner.py`: Módulo encargado de realizar los pings.
- `utils.py`: Módulo para el procesamiento y formato de resultados.
